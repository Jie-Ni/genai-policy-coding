# Data Boundary

This directory contains aggregate derived results for the PFE manuscript.

It intentionally does not contain verbatim university policy text. The source
documents were public institutional web pages and policy PDFs, but bulk
redistribution of their text is not assumed to be permitted.

Files:

- `pfe_measurement_summary.csv`: inter-model reliability and expert-validation
  metrics reported in the manuscript.
- `pfe_country_icc.csv`: country-level random-intercept signal reported in the
  manuscript.
- `pfe_regional_contrasts.csv`: locked region-level counts and p-values for the
  two headline regional contrasts.
- `pfe_temporal_drift.csv`: paired 64-institution temporal drift results.

The CSVs are aggregate result tables, not a substitute for the full internal
institution-level coding matrix.
